# Traceless / Meta Data

Traceless is a privacy-focused mobile application designed to help users detect and remove metadata from files before sharing them.

The project was built as part of a student research initiative focused on data protection, digital responsibility, and safer information sharing.

## Problem

Files such as images, documents, and media can contain hidden metadata. This metadata may reveal sensitive information such as device details, location data, author names, timestamps, or editing history.

Many users share files without knowing what hidden information they may expose.

## Goal

The goal of Traceless is to make metadata privacy easier to understand and more accessible to non-technical users.

## Main Features

- Detect metadata inside files
- Help users understand potential privacy risks
- Remove metadata before sharing
- Provide a simple mobile-first experience
- Promote safer digital habits

## Tech Stack

- Flutter
- Dart
- Mobile application development
- Privacy-by-design approach
